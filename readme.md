## How to develop

To run aplication locally:
* Download [Docker](https://www.docker.com/)
* Clone the repository
```
git clone https://github.com/sztwiorokr/webb.git
```
* Run in the repository dir
```
docker-compose up
```
You are ready to work. *Live Debugging is enabled.*
